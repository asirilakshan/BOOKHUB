# 📚 BookHub - Online Book Marketplace

## Overview

BookHub is a web-based application developed to provide an online platform for buying, selling, and renting books. The application allows users to explore available books, manage purchases and rentals, and interact with the system through a simple and responsive user interface.

This project was developed using PHP and MySQL as part of academic learning in web application development.

---

## Features

### User Management
- User Registration
- User Login and Logout
- Manage User Profile

### Book Management
- Browse Available Books
- Search Books
- View Book Details
- Buy Books Online
- Rent Books

### Shopping Cart
- Add Books to Cart
- Update Cart Items
- Remove Items from Cart
- Manage Book Purchases and Rentals

### Order Management
- Place Orders
- View Order Details
- Manage Book Transactions

### Contact System
- Submit Contact Messages
- Customer Communication Support

### Admin Dashboard
- Manage Users
- Manage Books
- Manage Orders
- Monitor System Activities

## How to Run

### Clone Repository

```
git clone https://github.com/asirilakshan
```

### Setup Project

Copy the project folder to:

```
C:\xampp\htdocs\BookHub
```

### Start XAMPP

Open XAMPP Control Panel and start:

- Apache
- MySQL

### Database Setup

1. Open phpMyAdmin
2. Create a database named:

```
bookhub
```

3. Import the provided SQL database file.

### Run Application

Open the browser and visit:

```
http://localhost/BookHub
```
